Heart Disease Dataset

About Dataset

Size : [303, 14]

Heart disease database contains 76 attributes, but all published experiments refer to using a subset of 14 of them. In particular, the Cleveland database is the only one that has been used by ML researchers to this date. The "goal" field refers to the presence of heart disease in the patient. It is integer valued from 0 (no presence) to 4. Learn more..

Attribute Description

Attributes Attr. Types Attr. inf Domain
age int64, Ratio age in year -
Sex int64, nominal Sex / gender 1 = Male; 0 = Female
Cp int64, nominal Chest pain type Value 1: typical angina; Value 2: atypical angina; Value 3: non-anginal pain; Value 4: asymptomatic
Trestbps int64, ratio Resting blood pressure in mm Hg on admission to the hospital
Chol int64, ratio serum cholestoral in mg/dl
Fbs int64, nominal fasting blood sugar > 120 mg/dl 1 = true; 0 = false
Restecg int64, order resting electrocardiographic results Value 0: normal; Value 1: having ST-T wave abnormality (T wave inversions and/or ST elevation or depression of > 0.05 mV); Value 2: showing probable or definite left ventricular hypertrophy by Estes' criteria
Thalach int64, ratio maximum heart rate achieved -
Exang int64, nominal exercise induced angina 1 = yes; 0 = no
Oldpeak float64, ratio ST depression induced by exercise relative to rest -
Slope int64, order the slope of the peak exercise ST segment Value 1: upsloping; Value 2: flat; Value 3: downsloping
Ca int64, order number of major vessels colored by flourosopy 0-3
Thal int64, order - 3 = normal; 6 = fixed defect; 7 = reversable defect
Target int64, nominal Status of patient from getting disease 1 = yes; 0 = no
